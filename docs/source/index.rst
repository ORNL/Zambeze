Zambeze documentation
=====================

Zambeze is a distributed task orchestration system to orchestrate science application tasks across the large-scale and edge computing systems.

The source code repository for Zambeze's Python package is available on `GitHub <https://github.com/ORNL/zambeze>`_. Report a bug using the repository's `Issues <https://github.com/ORNL/zambeze/issues>`_ page. Feature requests, questions, and other comments can be made on the `Discussions <https://github.com/ORNL/Zambeze/discussions>`_ page. The Zambeze developers can be reached via email at support@zambeze.org.
