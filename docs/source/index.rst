Zambeze documentation
=====================

Zambeze is a distributed task orchestration system to orchestrate science application tasks across the large-scale and edge computing systems.

The source code for Zambeze's Python package is available on `GitHub <https://github.com/ORNL/zambeze>`_. Our preferred channel to report a bug or request a feature is via Zambeze's Github `Issues Track <https://github.com/ORNL/zambeze/issues>`_.

You can also reach the Zambeze team via our support email support@zambeze.org.
