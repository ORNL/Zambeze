Zambeze Documentation
=====================

Welcome to the documentation for the zambeze Python pacakage. Zambeze is a distributed task orchestration system to orchestrate science application tasks across large-scale and edge computing systems.

The source code for zambeze is available on `GitHub <https://github.com/ORNL/zambeze>`_. Report a bug using the repository's `Issues <https://github.com/ORNL/zambeze/issues>`_ page. Feature requests, questions, and other comments can be made on the `Discussions <https://github.com/ORNL/Zambeze/discussions>`_ page. The zambeze developers can be reached via email at support@zambeze.org.
